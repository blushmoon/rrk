About Converged Access Edge Platform (CAEP)
The CAEP management platform provides complete functions required for edge applications deployment through a visual interface. Allow system integrators to use the shortest time to enhance the competitiveness of existing applications.

Customers can easily complete the deployment of edge nodes and edge applications through the CAEP management interface, as well as network application settings and maintenance. The visualized and user friendly management interface of the CAEP platform allows users to easily grasp the entire network configuration and application deployment. Use the platform's monitoring and instant alarm functions to control the status of each node to each service, reducing the risk of any service downtime.
